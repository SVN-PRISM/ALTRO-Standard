# ALTRO Core System

Система транскреации ALTRO - прототип Telegram Mini App.

## Технологии

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте http://localhost:3000 в браузере.

## Структура проекта

- `src/config/` - Конфигурация доменов ALTRO
- `src/core/` - Ядро валидации (Homonym & Stress Engine)
- `src/components/` - UI компоненты
- `src/app/` - Страницы Next.js
