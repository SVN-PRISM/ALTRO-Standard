# ALTRO CORE STANDARD (SVN-OPR)

## 1. Определение Ядра (Принцип фильтрации)
Контент перерабатывается LLM под управлением ALTRO по следующей цепочке:
- **Golden Standard:** 8 слов (характеристики) пропущенных через 5 внутренних доменов (Semantic, Context, Intent, Imagery/Metaphor, Ethical/Sacred). Создает первичный импульс.
- **Интерфейс управления:** 13 внешних доменов. Веса (слайдеры) определяют форму и влияние.
- **OPR Регулятор:** Проверка целостности и логики.
- **LLM (Local Qwen2.5 14b):** Исполнитель транскреации.

## 2. Двойное состояние OPR (Object-Predicate Relations)
- **Метафизика:** Резонанс Присутствия Наблюдателя (Архитектора).
- **Технология:** Контроль связей объекта и предиката, сохранение ударений и омонимов.

## 3. Директива для Composer:
Любое изменение кода должно соответствовать этому стандарту. Приоритет — сохранение Смыслового Суверенитета автора.
